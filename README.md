# Boilerplate for static website

* PostCSS(CSS Next)
* TypeScript
* Browser-sync
* StyleLint
* ESLint
* Prettier

# Usage

## Installation

```
cd path/to/project
npm i
```

## Development

```
npm run start
```

Start browser-sync and watch files

## Build

```
npm run build
```
* Cleaning `dist`
* Lint js files and css files.
* Format all files.
* Generate or copy files from `src` to `dist`.

## Pre-comit hock

* Type check with tsc
* Lint with ESLint and StyleLint
* Format with prettier
